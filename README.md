A simple REST API built with Java and Spring Boot, inspired by the official Swagger Petstore example.

## Technologies
- Java 21
- Spring Boot 3
- Spring Data JPA
- H2 Database
- Springdoc OpenAPI (Swagger)
- Lombok

## Features
- Create, read, update and delete pets
- Swagger UI for automatic API documentation
